# Number Cards (PWA)

A small number-guessing game that works offline, built with React, TypeScript, and Tailwind CSS.

The idea came from a game in a Christmas cracker (Christmas 2023).

Pick a secret number (default range 1—63). For each "card", answer **Yes/No** to "Is your number on this card?". At the end, the app reveals your number by adding together the hidden values of every card you said **Yes** to.

## Live demo

[Play Number Cards](https://karl-horning.github.io/number-cards-pwa/)

## Installation

```bash
git clone https://github.com/Karl-Horning/number-cards-pwa.git
cd number-cards-pwa
npm install
npm run dev
```

## Scripts and usage

| Command           | Description                          |
| ----------------- | ------------------------------------ |
| `npm run dev`     | Start local development server       |
| `npm run build`   | Build production assets              |
| `npm run preview` | Preview the production build locally |
| `npm run lint`    | Run ESLint                           |
